# Infinite-Star-Station-CreativeFiles
存储一些我的创意文件，永恒寒星网站的附属产品，看看大家需不需要 | 

I’m sharing some of my creative files—byproducts of the Eternal Cold Star website—to see if there is any interest.

注意事项： 代码极其不稳定，请勿将css和 html文件分离；需要进行大量编辑才能够投入稳定使用

目前的所有文件和 网页文件仅限本地使用，不建议在服务器上运行

文件基本介绍：
PIIN_final 是图片图库查看器

在库存页面上传图片即可将图片添加到网页图库，
可以提供图片的标题以及当前这一刻的想法，以及图片下方的一句话，用于做图片收藏记录
收藏的图片在添加图片的库存页面会自动按照月份，时间来排序，让你清楚知道每个图片的年份
这些信息都会保存在local storage内部，
之后，进入观察页即可看到所有的图片，按住左右方向键切换
在下方可以为每个图片撰写专属小说，记录你的灵感

【小细节：网站内部的色调会根据图片的整体色调改变】

infinite_star_final 是一个模仿linux, macos 系统的 一种特别的仿系统的浏览器主页导航页
可以在空闲时间，看一看这个页面，打开网页自带的电台音乐，感受片刻的平静。
可以使用的功能是 
1. 番茄时钟（管理时间）
2. 氛围信息显示模块（天气没有提供api,仅供观赏）
3. 简单的日历式日期查看系统，
4. 顶部链接标签栏，
5. 个人首选网站链接启动项
6. 时间栏，
7. 以及dock折叠栏
我结合了linux高度自定义的特点，拥有：
1. 自定义图片壁纸
   a. 调整饱和度
   b. 亮度
   c. 背景模糊程度
2. 自定义纯色壁纸的多种自定义选项
3. 分为多种颜色模式（包括，初始蓝色，自带模版，自适应壁纸）的音频条自定义选项
4. 桌面背景文字自定义
5. brand文字自定义
6. 元素显示控制功能 （是否开启或者关闭特定元素）
7. 导出导入配置功能

我也结合了 mac的设计理念以及基本的壁纸功能，包括但不限于：
1. 根据时间自己变化的壁纸
2. 专注于用户体验的ui
3. 以及Dock 栏的样式

以上的所有自定义功能和设置选项也存储于 localstorage内部，需要通过导出设置来保存，否则会丢失

vocab_notebook_fianl 是一个 单词测试软件，是方便我进行单词背诵的
本项目 以便签为灵感，采用对照背诵的方式完成单词测试的目的
使用说明： 
      分为单词库和 错题本两大模块， 
      词库功能有：
       - 切换词库
       - 新建词库
       - 删除以及重命名词库
       可以
       - 添加单词的原词和中文释义
       - 可以批量导入单词
      
      导入单词完成后，
      在左侧栏目左下角单词列表滑动查看单词，默认的单词不会显示意思，需要点击空白部分才可以暂时查看释义
      单词列表可以排序，在手动测试上方的排序选项可以选择
       - 以添加顺序排序
       - 以已测试顺序排序
       - 或者未测试顺序排序
      
      单词列表也可以编辑相应已经添加好的单词
      
      左侧栏单词表上方是测试的功能，测试可以选择 整齐排列或者分散排列
      整齐排列是 Grid 表格布局而分散排列是 模仿单词表和便签式的测试，保证不是假测试
      
      测试模式分为 手动测试和 随机测试， 
      随机测试可以选择测试单词数量，随机抽取单词。
      手动测试需要点击单词表每个单词前的框选中才能将单词添加进测试列表进行手动测试
      
      每次测试完毕，需要自己判分，最终，等待测试进度完毕即总单词 （手动测试是你选择的单词， 随机测试是所有单词）全部测试完毕，即可显示你的分数，错的单词以及单词各自的解释，此时可以将错的单词添加到错题本。
      
      所有已存储的数据均在localstorage,所以需要通过导出存档和 导入存档有目的性的备份数据，防止丢失。

flat_note_wall_inbox 是一个特别的便签式备忘录网页

网页结构分为三个部分，
- 第一部分： 便签墙
- 第二部分： 聊天框式记录仪
- 第三部分： 链接收集器


*便签墙:*
便签墙可以随时储存你的信息和想法
*聊天框式记录仪：*
聊天框式记录仪模拟微信的聊天功能，你可以在里头自己给自己发消息，以此完成记录的目的
它可以记录你每次发送消息的时间， 如果遇到单个链接信息条还会变成 方便查看的链接面板。
*便签墙和聊天框式记录仪都在左侧，各自在上下方，可以通过拖拽聊天框式记录仪的上方线条来拉伸改善他们的形状，起到方便查看聊天框信息的作用。*

<----------->

*链接收集器：*
每次发送的单个链接信息条都会被记录在链接收集器里，方便再次查看，免除了上下翻聊天记录的问题

注： 所有的元素，包括聊天条，链接收集器的链接和 便签都可以删除。 为了免除让画面分散注意力，之前的单词测试其和备忘录记录网页都保持了方格纸的风格。
