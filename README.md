# PIIN Final

图片图库查看器。

### 功能

- 上传图片到图库
- 为图片记录标题、想法与说明
- 自动按照时间排序
- 为每张图片撰写专属故事
- 根据图片整体色调自动调整页面主题

> 所有数据保存在 Local Storage。
